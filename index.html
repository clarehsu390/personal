Your Aesthetic

Your Aesthetic is a fullstack single-page web application inspired by Pinterest. Created using Ruby on Rails, React.js and Redux. Users can add pins to boards to save images.

Word Frenzy

Word Frenzy is a browser game, handling complicated mouse events. Created using vanilla Javascript and jQuery. Implemented dictionary lookup using Trie data structure.

Clustering Visualization

Clustering Visualization provides visual representation of K-means clustering, a simple algorithm used in machine learning. Created using D3.js and Django.

About Me

My name is Clare Hsu. I graduated from UC Davis with a degree in linguistics but discovered I wanted to accomplish more with this degree. A professor of mine suggested computational linguistics, so I decided to fully dedicate myself to software engineering.
I attended App Academy to develop myself and gain the experience and skills. I enjoy solving problems and thinking critically. In the future, I want to pursue a career in natural language processing. In my free time, I love walking my dogs and discovering new food spots in San Francisco.
